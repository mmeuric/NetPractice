**NetPractice** is a networking project from the 42 curriculum.
Its goal is to teach the fundamentals of **IP addressing**, **routing**, **subnetting**, and basic network troubleshooting through 10 practical levels.

Each level presents a broken virtual network that must be fixed by adjusting IPs, masks, routes, gateways, and interfaces until the entire topology becomes fully functional.

This project helped me solidify core networking concepts like:

* Subnetting & CIDR
* Host and network ranges
* Router configuration logic
* Gateway selection & packet flow
* Local vs. remote addressing rules
